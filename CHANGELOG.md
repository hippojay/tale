## [1.0.0] - 2026-03-07

### Added
- Initial release of TALE Method (The Author's Literary Engine)
- 12 specialized literary agents across 5 domains
- 60+ guided workflows for complete writing lifecycle
- **Establish Narrative Voice** workflow for proactive voice definition
- **Prose Quality Audit** workflow with quantitative metrics (TTR, sentence analysis)
- AI-generic pattern detection across 64 patterns
- Enhanced **Continuity Check** workflow for systematic story consistency
- Prose sophistication system (accessible/polished/literary/experimental)
- Genre-adaptive guidance and workflow recommendations
- Party Mode for multi-agent collaboration
- Advanced Elicitation for deep requirements gathering
- Story Bible as single source of truth
- Complete phase system: Planning → Writing → Review → Polish → Publishing

### Workflows by Phase
- **Phase 1: Planning** — 20 workflows (story planning, character development, world-building)
- **Phase 2: Writing** — 15 workflows (writing production, narrative enrichment)
- **Phase 3: Review** — 13 workflows (story review, developmental editing, quality audit)
- **Phase 4: Polish** — 13 workflows (line editing, copy editing)
- **Phase 5: Publishing** — 5 workflows (market analysis, query letters, launch planning)
- **Anytime** — 5 core utilities (help, brainstorming, party mode, document tools)

### Built On
- Adapted from BMAD Method framework for software development
- Inherits agent orchestration, party mode, and structured elicitation
- Customized for literary domain with genre-adaptive intelligence
