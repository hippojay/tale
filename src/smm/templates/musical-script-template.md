> **FORMAT GUIDE — Industry Standard Libretto Format**
>
> This template demonstrates industry-standard musical theatre script (libretto) format.
> Delete this guide block before production use.
>
> **Act/Scene headings**: Centered, all caps
> **Stage directions**: Present tense. Parenthetical or italic. Describe only what is visible or audible to an audience. Keep lean — trust directors and designers.
> **Character names (before dialogue)**: ALL CAPS, left-aligned
> **Dialogue**: Regular prose, left-aligned under character name
> **Parentheticals (tone/action)**: Indented in parentheses. Use sparingly — only when meaning is ambiguous without them.
> **Song cues**: Centered. Song title in ALL CAPS or quotes. Followed by a brief stage direction.
> **Lyrics**: Indented under the singing character's name (formatted like dialogue but set apart visually).
> **Never use camera directions**: This is stage, not screen.

*(A moment. CHARACTER A looks away.)*
