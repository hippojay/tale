# Create Character Profiles Workflow

## Objective
Create comprehensive character profiles for main and supporting characters.

## Prerequisites
- SRD with story concept
- Story Bible initialized

## Steps

### 1. Gather Requirements
- Review SRD for character needs
- Identify protagonist, antagonist, supporting roles
- Note genre expectations for character types

### 2. Build Core Profile
For each main character:
- Name and role in story
- Age, physical description
- Personality traits (strengths, flaws)
- Background and history
- Goals and motivations
- Internal conflict/wound
- External obstacles

### 3. Develop Psychology
- Core belief/worldview
- Fear and desire
- How they change under pressure
- Blind spots and biases

### 4. Establish Voice
- Speech patterns and vocabulary
- Verbal tics or catchphrases
- How they differ from other characters

### 5. Map Relationships
- Connection to other characters
- Power dynamics
- Conflict sources
- Alliance potential

### 6. Document Arc
- Starting state
- Key transformation moments
- Ending state
- What they learn

### 7. Update Story Bible
- Add all character details
- Cross-reference relationships
- Note any world-building implications

## Output
- Character profiles in Story Bible
- Separate character sheets if detailed: `{output_folder}/characters/{name}.md`
