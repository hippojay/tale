# Design Character Arcs Workflow

## Objective
Map character transformation arcs aligned with plot progression.

## Prerequisites
- Character profiles created
- Story structure outlined

## Steps

### 1. Define Arc Type
- Positive change (growth)
- Negative change (fall)
- Flat arc (tests existing belief)
- Circular (returns transformed)

### 2. Establish Starting Point
- Character's false belief or limitation
- Current emotional state
- Relationships at start
- What they want vs. what they need

### 3. Map Transformation Beats
- Inciting incident impact
- First threshold crossing
- Midpoint revelation
- Dark night of the soul
- Climactic choice
- Resolution state

### 4. Align with Plot
- Which plot events trigger growth?
- How does character change affect plot?
- Where are the resistance points?
- What forces change?

### 5. Track by Chapter
Create arc tracker:
| Chapter | Event | Internal Shift | External Change |
|---------|-------|----------------|-----------------|

### 6. Verify Coherence
- Is change earned and believable?
- Are there enough catalyst moments?
- Does ending satisfy arc promise?

### 7. Update Story Bible
- Document arc in character section
- Add to timeline tracking
- Note key scenes for arc

## Output
- Character arc document
- Updated Story Bible with arc details
