# Develop Dialogue Voice Workflow

## Objective
Create distinct, authentic dialogue voices for each character.

## Prerequisites
- Character profiles completed
- Character backgrounds established

## Steps

### 1. Analyze Character Background
- Education level → vocabulary complexity
- Regional origin → dialect/idioms
- Profession → jargon and references
- Age → generational speech patterns
- Personality → communication style

### 2. Define Speech Patterns
- Sentence length tendency (short/long)
- Formal vs. casual register
- Direct vs. indirect communication
- Questions vs. statements ratio
- Interruption patterns

### 3. Create Verbal Signatures
- Catchphrases or repeated expressions
- Filler words (um, like, well)
- Unique word choices
- Metaphor preferences
- Humor style

### 4. Establish Emotional Variations
- How voice changes under stress
- Anger expression style
- Joy expression style
- Fear expression style
- Deception tells

### 5. Differentiation Test
- Can you identify speaker without tags?
- Do characters sound different from each other?
- Is voice consistent across scenes?

### 6. Write Sample Dialogue
- Create test conversation between characters
- Verify distinctiveness
- Refine as needed

### 7. Document in Story Bible
- Add voice details to character section
- Create quick-reference voice guide
- Note any dialect spelling conventions

## Output
- Voice profiles in Story Bible
- Sample dialogue for reference
