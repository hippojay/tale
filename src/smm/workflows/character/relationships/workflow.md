# Character Relationships Workflow

## Objective
Map and develop relationships between characters.

## Prerequisites
- Character profiles created
- Story structure outlined

## Steps

### 1. Create Relationship Matrix
| Character | Protagonist | Antagonist | Support A | Support B |
|-----------|-------------|------------|-----------|-----------|
| Protagonist | - | | | |
| Antagonist | | - | | |

### 2. Define Each Relationship
- Nature (family, romantic, professional, adversarial)
- History (how they met, shared past)
- Current state (allies, enemies, strangers)
- Power dynamic (equal, dominant, submissive)
- Emotional bond (love, hate, indifference, fear)

### 3. Identify Conflict Sources
- Competing goals
- Past wounds
- Misunderstandings
- Value differences
- Resource competition

### 4. Map Relationship Arcs
- Starting state
- Key tension points
- Transformation moments
- Resolution state

### 5. Plan Key Scenes
- First meeting/introduction
- Conflict escalation scenes
- Bonding moments
- Betrayal or reconciliation
- Final state scene

### 6. Check Story Function
- Does relationship serve plot?
- Does it reveal character?
- Does it support theme?
- Is it necessary?

### 7. Update Story Bible
- Add relationship details
- Cross-reference between characters
- Note key relationship scenes

## Output
- Relationship map/matrix
- Updated Story Bible
