# Copy Edit Chapter Workflow

## Objective
Complete copy edit for correctness and consistency.

## Steps

### 1. Grammar Pass
- Subject-verb agreement
- Tense consistency
- Pronoun clarity
- Modifier placement

### 2. Punctuation Pass
- Comma usage
- Dialogue punctuation
- Apostrophes
- Semicolons/colons

### 3. Spelling Pass
- Spell check
- Homophones
- Character/place names
- Specialized terms

### 4. Consistency Pass
- Style choices consistent
- Formatting uniform
- Numbers/dates standardized

### 5. Fact Check
- Verify checkable facts
- Timeline accuracy
- Internal consistency

### 6. Query List
- Flag unclear passages
- Note potential issues
- Questions for author

## Output
- Copy-edited chapter
- Query list
- Style sheet updates
