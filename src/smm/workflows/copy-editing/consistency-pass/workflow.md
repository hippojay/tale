# Consistency Pass Workflow

## Objective
Check consistency of style, terminology, and formatting across the full manuscript.

## Steps

### 1. Establish Baseline
- Load style sheet (or create one if missing)
- Note established conventions from early chapters
- Document character name spellings, place names, special terms

### 2. Character Consistency
- Names spelled identically throughout
- Titles and honorifics consistent
- Physical descriptions don't contradict
- Age references align with timeline

### 3. Terminology Consistency
- Specialized terms used uniformly
- World-building terms capitalized consistently
- Abbreviations introduced and used consistently
- Slang and dialect patterns maintained

### 4. Formatting Consistency
- Chapter heading format uniform
- Scene break markers consistent
- Emphasis (italics/bold) used consistently
- Dialogue formatting standardized

### 5. Style Consistency
- Number formatting (spelled out vs. digits)
- Date and time formatting
- Hyphenation patterns
- Oxford comma usage consistent

### 6. Cross-Reference Check
- Internal references accurate
- Foreshadowing elements followed through
- Callbacks reference correct earlier events

### 7. Document Findings
- List all inconsistencies found
- Categorize by severity
- Recommend standardization choices

## Output
- Consistency report with specific locations
- Updated style sheet
- Recommended fixes
