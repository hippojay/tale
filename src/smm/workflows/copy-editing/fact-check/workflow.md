# Fact Check Workflow

## Objective
Verify all checkable facts, references, and internal logic throughout the manuscript.

## Steps

### 1. Identify Checkable Claims
- Historical dates and events
- Geographic details (distances, locations, landmarks)
- Scientific or technical references
- Cultural references and customs
- Legal or procedural details

### 2. Internal Fact Check
- Timeline consistency (days, seasons, character ages)
- Distance and travel time plausibility
- Character knowledge matches what they've witnessed
- Cause and effect chains hold up logically

### 3. External Fact Check
- Verify real-world references against sources
- Check historical accuracy where claimed
- Validate technical details
- Confirm cultural details are accurate and respectful

### 4. World-Building Fact Check (Fiction)
- Magic/technology system rules followed consistently
- World rules established early are honored throughout
- No contradictions in world-building details
- Consequences of world rules play out logically

### 5. Dialogue Fact Check
- Characters don't reference events they weren't present for
- Knowledge revealed in dialogue matches timeline
- Technical dialogue is accurate to character expertise

### 6. Document Findings
- Flag errors with specific locations
- Distinguish factual errors from creative license
- Suggest corrections where possible
- Query author on ambiguous items

## Output
- Fact-check report with page/chapter references
- Error list categorized by type
- Query list for author decisions
