# Style Sheet Workflow

## Objective
Create comprehensive style sheet for manuscript.

## Steps

### 1. Character Names
- Full names and nicknames
- Spelling variations
- Titles and honorifics

### 2. Place Names
- Locations and spelling
- Fictional place conventions
- Real place accuracy

### 3. Terminology
- Made-up words
- Technical terms
- Foreign words

### 4. Style Choices
- Numbers (spelled/numerals)
- Time format
- Date format
- Hyphenation decisions

### 5. Formatting
- Chapter headings
- Scene breaks
- Emphasis (italics/bold)
- Dialogue conventions

### 6. Timeline
- Key dates
- Character ages
- Event sequence

## Output
- Complete style sheet document
