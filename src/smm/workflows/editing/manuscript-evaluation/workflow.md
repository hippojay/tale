# Manuscript Evaluation Workflow

## Objective
Comprehensive developmental edit of complete manuscript.

## Prerequisites
- Complete manuscript draft
- Story Bible for reference
- SRD for intent

## Steps

### 1. Initial Read
- Read manuscript completely
- Note overall impression
- Identify emotional response
- Mark problem areas for return

### 2. Story Promise Evaluation
- Does opening make a promise?
- Does ending fulfill it?
- Is the journey satisfying?

### 3. Structure Assessment
- Are act breaks clear?
- Is pacing appropriate?
- Does tension escalate?
- Is climax earned?
- Is resolution complete?

### 4. Character Evaluation
- Are protagonists compelling?
- Are antagonists credible?
- Are arcs complete?
- Is motivation clear?
- Are voices distinct?

### 5. Plot Analysis
- Are there plot holes?
- Is logic consistent?
- Are coincidences earned?
- Is causality clear?
- Are stakes maintained?

### 6. Theme Assessment
- Is theme present?
- Is it organic or forced?
- Does it resonate?
- Is it consistent?

### 7. World-Building Check
- Is world consistent?
- Is it immersive?
- Are rules followed?
- Is detail balanced?

### 8. Genre Compliance
- Does it meet genre expectations?
- Where does it excel?
- Where does it fall short?

### 9. Prioritize Issues
- **Critical:** Story-breaking problems
- **Major:** Significant quality issues
- **Minor:** Polish items

### 10. Create Editorial Report
- Executive summary
- Strengths to preserve
- Issues by priority
- Specific recommendations
- Revision roadmap

## Output
- Comprehensive editorial report
- Prioritized revision list
- Revision roadmap
