# Revision Roadmap Workflow

## Objective
Create prioritized revision plan from editorial feedback.

## Prerequisites
- Editorial report or feedback
- Manuscript draft
- Story Bible

## Steps

### 1. Gather All Feedback
- Compile all editorial notes
- Include reviewer feedback
- Add author concerns
- Note beta reader input

### 2. Categorize Issues
**Structural:** Plot, pacing, arc
**Character:** Motivation, consistency, voice
**World:** Setting, rules, immersion
**Craft:** Prose, dialogue, description
**Continuity:** Timeline, facts, logic

### 3. Prioritize by Impact
**Priority 1 - Critical:**
- Story-breaking issues
- Must fix for story to work
- Examples: plot holes, missing climax, unclear motivation

**Priority 2 - Major:**
- Significant quality issues
- Story works but weakened
- Examples: pacing problems, flat characters, weak opening

**Priority 3 - Minor:**
- Polish items
- Good to great improvements
- Examples: prose refinement, dialogue sharpening

### 4. Sequence Revisions
Order matters:
1. Structural changes first (may eliminate other issues)
2. Character work second
3. Scene-level revisions third
4. Line-level polish last

### 5. Create Revision Passes
**Pass 1: Structure**
- Major plot changes
- Scene additions/deletions
- Reordering

**Pass 2: Character**
- Arc completion
- Motivation clarity
- Voice consistency

**Pass 3: Scenes**
- Pacing adjustments
- Tension enhancement
- Transition smoothing

**Pass 4: Polish**
- Prose refinement
- Dialogue sharpening
- Description balance

### 6. Estimate Effort
For each item:
- Time estimate
- Difficulty level
- Dependencies

### 7. Create Checklist
Actionable items with:
- Specific task
- Location (chapter/scene)
- Priority level
- Completion checkbox

### 8. Set Milestones
- Pass completion dates
- Review checkpoints
- Final deadline

## Output
- Prioritized revision roadmap
- Pass-by-pass checklist
- Timeline with milestones
