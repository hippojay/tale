# Sensory Audit Workflow

## Objective
Audit and balance sensory engagement across chapter.

## Prerequisites
- Chapter to audit
- Story Bible sensory palette

## Steps

### 1. Catalog Existing Senses
Mark each sensory reference:
- 👁️ Visual
- 👂 Auditory
- 👃 Olfactory
- ✋ Tactile
- 👅 Gustatory

### 2. Calculate Distribution
- Count each sense type
- Note which scenes have which
- Identify gaps

### 3. Assess Balance
Common problem: Over-reliance on visual
- Is there variety?
- Are non-visual senses present?
- Does balance fit scene type?

### 4. Scene-Appropriate Senses
- Action scenes: tactile, auditory
- Intimate scenes: all senses
- Tension scenes: auditory, tactile
- Memory scenes: olfactory triggers

### 5. Identify Enhancement Points
- Where to add sound?
- Where to add smell?
- Where to add touch?
- Where taste fits naturally?

### 6. Add Missing Senses
- Integrate naturally
- Connect to POV experience
- Don't force or overload

### 7. Quality Check
- Sensory details specific?
- Clichés avoided?
- Details serve story?

### 8. Final Balance Review
- Variety achieved?
- Immersion improved?
- Pacing maintained?

## Output
- Sensory audit report
- Enhanced chapter with balanced senses
