# Line Edit Chapter Workflow

## Objective
Complete line edit of chapter for prose quality.

## Steps

### 1. First Read
- Read for overall flow
- Note voice patterns to preserve
- Mark problem areas

### 2. Sentence-Level Edit
- Tighten wordy sentences
- Strengthen weak verbs
- Eliminate redundancy
- Clarify confusing passages

### 3. Rhythm Check
- Vary sentence length
- Check paragraph flow
- Ensure natural reading rhythm

### 4. Word Choice
- Replace weak words
- Eliminate clichés
- Strengthen imagery

### 5. Dialogue Polish
- Tighten exchanges
- Sharpen character voice
- Reduce tags

### 6. Final Pass
- Read aloud for flow
- Verify voice preserved
- Check consistency

## Output
- Line-edited chapter
- Edit notes for author
