# Description Refinement Workflow

## Objective
Refine descriptive passages for clarity, impact, and purpose.

## Steps

### 1. Identify Descriptive Passages
- Mark all description blocks (setting, character, action)
- Note length and placement of each
- Flag passages that halt narrative momentum

### 2. Purpose Check
- Every description must serve story, character, or mood
- Cut descriptions that exist only to describe
- Ensure setting details reveal something about characters or conflict

### 3. Specificity Pass
- Replace generic details with specific, evocative ones
- "A tree" becomes "a wind-bent oak" when it matters
- Choose details that do double duty (setting + mood)
- Remove details that don't earn their place

### 4. Sensory Balance
- Check for over-reliance on visual description
- Add sound, smell, touch, taste where natural
- Match sensory focus to POV character's awareness
- Use sensory details to ground emotional moments

### 5. Metaphor and Simile Audit
- Remove clichéd comparisons
- Ensure metaphors illuminate rather than obscure
- Check for mixed metaphors
- Verify figurative language matches tone and setting

### 6. Integration Check
- Description woven into action, not dumped in blocks
- Characters interact with setting, not just observe it
- Pacing maintained through descriptive passages
- Transitions in and out of description feel natural

## Output
- Refined descriptive passages
- Edit notes explaining significant changes
