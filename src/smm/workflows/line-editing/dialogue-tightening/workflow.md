# Dialogue Tightening Workflow

## Objective
Tighten and sharpen dialogue for maximum impact and authenticity.

## Steps

### 1. Identify Dialogue Patterns
- Read all dialogue in chapter/section
- Note each character's speech patterns
- Flag dialogue that sounds generic or interchangeable

### 2. Cut Filler
- Remove greetings and pleasantries that don't serve story
- Cut "on the nose" dialogue (characters saying exactly what they feel)
- Eliminate redundant exchanges
- Remove dialogue that restates what narration already showed

### 3. Sharpen Subtext
- Ensure characters talk around difficult topics realistically
- Add tension through what's unsaid
- Let actions contradict words where appropriate
- Layer meaning beneath surface conversation

### 4. Differentiate Voices
- Each character should sound distinct
- Vocabulary matches education and background
- Speech rhythm reflects personality
- Verbal tics used sparingly but consistently

### 5. Tighten Exchanges
- Shorten long speeches into natural back-and-forth
- Cut dialogue tags where speaker is clear from context
- Replace said-bookisms with action beats
- Ensure pacing matches scene tension

### 6. Read Aloud Test
- Read dialogue exchanges aloud
- Flag anything that sounds unnatural
- Verify rhythm feels like real conversation

## Output
- Tightened dialogue with edit notes
- Character voice reference notes
