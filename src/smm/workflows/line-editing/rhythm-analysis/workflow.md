# Rhythm Analysis Workflow

## Objective
Analyze and improve prose rhythm for natural, engaging reading flow.

## Steps

### 1. Sentence Length Mapping
- Map sentence lengths across the chapter
- Identify monotonous stretches (all similar length)
- Flag sections where rhythm doesn't match content

### 2. Paragraph Flow
- Check paragraph length variation
- Ensure short paragraphs used for impact
- Long paragraphs reserved for immersive moments
- Single-line paragraphs used sparingly for punch

### 3. Scene Rhythm
- Action scenes: shorter sentences, punchy verbs, quick paragraphs
- Emotional scenes: varied length, breathing room, sensory detail
- Dialogue scenes: natural back-and-forth rhythm
- Reflective scenes: longer, flowing sentences with internal cadence

### 4. Repetition Audit
- Flag unintentional word repetition within paragraphs
- Check for repeated sentence structures
- Identify overused sentence openers (especially "He/She" starts)
- Note intentional repetition for rhetorical effect

### 5. Transition Flow
- Scene transitions smooth and purposeful
- Chapter openings establish rhythm quickly
- Chapter endings land with appropriate cadence
- Time jumps handled without jarring the reader

### 6. Read Aloud Test
- Read problematic sections aloud
- Mark where breath feels unnatural
- Note where emphasis falls — is it where intended?
- Flag tongue-twisters or awkward sound combinations

## Output
- Rhythm analysis report with specific locations
- Suggested revisions for problem areas
- Before/after examples for key fixes
