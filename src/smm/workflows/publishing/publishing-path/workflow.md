# Publishing Path Workflow

## Objective
Evaluate traditional vs. self-publishing to determine the best path for this project.

## Steps

### 1. Project Assessment
- Genre and market category
- Target audience and their buying habits
- Author's goals (career vs. single project)
- Timeline expectations

### 2. Traditional Publishing Evaluation
- Genre fit for traditional market
- Comp titles recently acquired by publishers
- Realistic advance expectations
- Timeline: query → agent → submission → publication (18-36 months)
- Pros: prestige, distribution, advance, editorial support
- Cons: loss of control, slow timeline, lower royalties

### 3. Self-Publishing Evaluation
- Genre viability in self-pub market
- Investment required (editing, cover, formatting, marketing)
- Realistic income expectations
- Timeline: manuscript ready → published (1-3 months)
- Pros: control, speed, higher royalties, rights retained
- Cons: upfront costs, all marketing on author, stigma (diminishing)

### 4. Hybrid Options
- Small press publishers
- Hybrid publishers (author invests, publisher distributes)
- Serial publishing platforms
- Kickstarter/crowdfunding models

### 5. Author Readiness Assessment
- Writing craft level
- Marketing willingness and ability
- Financial resources for self-pub
- Patience for traditional timeline
- Long-term career vision

### 6. Recommendation
- Best fit path with rationale
- Specific next steps for chosen path
- Backup plan if primary path doesn't work
- Timeline and milestones

## Output
- Publishing path analysis report
- Recommended path with rationale
- Action plan with next steps
