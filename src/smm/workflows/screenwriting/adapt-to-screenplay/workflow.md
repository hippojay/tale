# Adapt to Screenplay Workflow

## Objective
Transform existing prose (novel, short story, chapter) or story concept into a properly formatted screenplay.

## Prerequisites
- Source material (prose manuscript, story outline, or detailed concept)
- Story Requirements Document (SRD) if available
- Story Bible for character and world reference

## Steps

### 1. Source Analysis
- Read complete source material
- Identify the core dramatic question
- Note which elements are inherently cinematic (action, dialogue, visual moments)
- Flag elements that won't translate (internal monologue, lengthy description, backstory)

### 2. Structural Mapping
- Map source material to three-act structure:
  - Act 1 (pp. 1-30): Setup, inciting incident
  - Act 2A (pp. 30-60): Rising complications, fun and games
  - Act 2B (pp. 60-90): Midpoint shift, stakes escalate
  - Act 3 (pp. 90-120): Climax, resolution
- Identify 8-sequence breakdown within acts
- Determine what to cut, compress, or combine
- Identify what needs to be invented (visual equivalents for prose exposition)

### 3. Scene List
- Create master scene list with:
  - Scene number
  - INT/EXT and location
  - Time of day
  - Characters present
  - Scene purpose (what changes by end of scene)
  - Estimated page count
- Verify total page count hits target (90-120 for feature, 45-60 for hour drama, 22-30 for half-hour)

### 4. Adaptation Decisions
For each major prose element, decide:
- **Keep as-is**: Dialogue or action that works cinematically
- **Visualize**: Convert internal thought to visible action or behavior
- **Compress**: Combine multiple scenes/chapters into one scene
- **Cut**: Remove entirely (doesn't serve screen story)
- **Invent**: Create new scenes to replace prose exposition

### 5. Draft Screenplay
Write in proper format:
```
FADE IN:

EXT. LOCATION - TIME OF DAY

Action lines in present tense. Brief, visual, 
three lines maximum per block.

                    CHARACTER NAME
          Dialogue goes here. Keep it tight.
          People don't speak in paragraphs.

                    CHARACTER NAME 2
                         (beat)
          Neither do they.
```

### 6. Adaptation Review
- Does the screen version stand alone without the source?
- Are all prose-dependent moments now visual?
- Does dialogue sound spoken, not written?
- Is pacing appropriate for screen (faster than prose)?
- Are characters introduced visually, not through exposition?

## Output
- Complete screenplay in standard format
- Adaptation notes (what changed from source and why)
- Scene list with page counts
