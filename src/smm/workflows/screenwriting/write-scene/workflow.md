# Write Screenplay Scene Workflow

## Objective
Write a single properly formatted screenplay scene.

## Prerequisites
- Scene context (what comes before and after)
- Characters involved
- Scene objective (what must change by scene's end)

## Steps

### 1. Scene Setup
- Determine scene heading: `INT./EXT. LOCATION - DAY/NIGHT`
- Identify POV character for the scene
- Clarify the conflict: what does the character want, what's in the way?
- Determine the turn: how is the situation different at scene's end?

### 2. Write Scene
Follow format rules:
- Action lines: present tense, what we SEE and HEAR only
- Max 3-4 lines per action block, then break
- Character name CENTERED AND CAPS on first introduction with brief visual description
- Dialogue centered under character name
- Parentheticals used sparingly (only when meaning would be misread without them)
- No camera directions (unless writing a shooting script)

### 3. Dialogue Check
- Read all dialogue aloud
- Each character should sound distinct
- Cut any line that states the obvious
- Add subtext: what are they really saying beneath the words?
- Trim — if a speech is more than 3 lines, it's probably too long

### 4. Visual Storytelling Check
- Is there a visual element carrying emotion? (not just people talking)
- Are characters doing something physical during dialogue?
- Does the setting contribute to mood?
- Could any dialogue be replaced with a look, gesture, or action?

### 5. Scene Economy
- Enter as late as possible
- Exit as early as possible
- Cut greetings, goodbyes, and small talk unless they reveal character
- Every line of action and dialogue must earn its place

## Output
- Formatted screenplay scene
- Scene notes (context, purpose, connections to surrounding scenes)
