# Write Screenplay Workflow

## Objective
Write an original screenplay from concept, outline, or beat sheet.

## Prerequisites
- Story concept, logline, or detailed outline
- SRD if available
- Target format: feature (90-120pp), hour drama (45-60pp), half-hour (22-30pp)

## Steps

### 1. Concept Lock
- Confirm logline: "When [inciting incident], [protagonist] must [goal] or else [stakes]"
- Identify genre and tone
- Determine target length and format
- Clarify audience (studio feature, indie, TV pilot, short)

### 2. Beat Sheet
Create beat sheet (Blake Snyder / Save the Cat structure):
- Opening Image (p.1)
- Theme Stated (p.5)
- Setup (pp.1-10)
- Catalyst / Inciting Incident (p.12)
- Debate (pp.12-25)
- Break into Two (p.25)
- B Story (p.30)
- Fun and Games (pp.30-55)
- Midpoint (p.55)
- Bad Guys Close In (pp.55-75)
- All Is Lost (p.75)
- Dark Night of the Soul (pp.75-85)
- Break into Three (p.85)
- Finale (pp.85-110)
- Final Image (p.110)

### 3. Scene Outline
- Write one-line description per scene
- Assign INT/EXT, location, time of day
- Note characters in each scene
- Estimate page count per scene
- Verify total hits target

### 4. Write Draft
For each scene:
- Scene heading: `INT./EXT. LOCATION - DAY/NIGHT`
- Action lines: present tense, visual, lean (3-4 lines max per block)
- Dialogue: subtext-driven, speakable, character-distinct
- Enter late, leave early
- End scenes on a turn or question

### 5. Page One Polish
- First 10 pages must hook — verify they do
- Protagonist introduced with a defining visual moment
- World established through action, not exposition
- Tone set clearly

### 6. Full Draft Review
- Read straight through for pace
- Check page count against target
- Verify every scene has conflict
- Ensure no "talking heads" scenes (add visual business)
- Confirm act breaks land at right page counts

## Output
- Complete screenplay draft in standard format
- Beat sheet
- Scene list
