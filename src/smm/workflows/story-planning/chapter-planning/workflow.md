# Chapter Planning Workflow

## Objective
Create detailed chapter-by-chapter outline with beats and objectives.

## Prerequisites
- Story structure completed
- Character arcs defined
- Story Bible initialized

## Steps

### 1. Determine Chapter Count
Based on target word count and genre:
- Short novel (60k): 15-20 chapters
- Standard novel (80k): 20-25 chapters
- Long novel (100k+): 25-35 chapters

### 2. Distribute Structure Beats
- Map act breaks to chapter numbers
- Place major plot points
- Position midpoint and climax
- Allocate subplot beats

### 3. Plan Each Chapter
For each chapter, define:
- **Chapter Goal:** What must be accomplished
- **POV Character:** Whose perspective
- **Opening Hook:** How it starts
- **Scenes:** 2-4 scenes with objectives
- **Conflict:** What tension drives the chapter
- **Character Beat:** Arc progression moment
- **Closing Hook:** Why reader continues

### 4. Scene Beat Details
For each scene:
- Location
- Characters present
- Scene objective
- Conflict/tension source
- Outcome/change
- Transition to next

### 5. Verify Pacing
- Action/reflection balance
- Tension escalation
- Breathing room placement
- Cliffhanger distribution

### 6. Check Arc Alignment
- Character arcs progress appropriately
- Subplot threads woven in
- Theme reinforced naturally

### 7. Create Chapter Outline Document
Format per chapter:
```
## Chapter X: {Title}
**Goal:** 
**POV:** 
**Word Count Target:** 

### Scene 1
- Location:
- Characters:
- Objective:
- Conflict:
- Outcome:

### Scene 2
...
```

### 8. Update Story Bible
- Add chapter summaries to timeline
- Note key events per chapter

## Output
- Complete chapter outline document
- Updated Story Bible timeline
