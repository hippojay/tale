# Character Arcs Workflow (Story Architect)

## Objective
Map character development arcs to story progression ensuring growth aligns with plot beats.

## Prerequisites
- Character profiles exist
- Story structure defined

## Steps

### 1. Identify Arc Characters
- Which characters have arcs?
- Protagonist (required)
- Antagonist (often)
- Key supporting (selective)

### 2. Define Arc Endpoints
For each arc character:
- **Start State:** Who are they at opening?
- **End State:** Who are they at closing?
- **Core Change:** What transforms?

### 3. Align with Plot Structure
Map arc beats to story beats:

| Story Beat | Plot Event | Character Arc Moment |
|------------|------------|---------------------|
| Inciting Incident | | Arc catalyst |
| First Plot Point | | Commits to change |
| Midpoint | | Revelation/shift |
| Dark Moment | | Faces core wound |
| Climax | | Proves change |
| Resolution | | New equilibrium |

### 4. Plan Transformation Scenes
- Which scenes show change?
- What forces growth?
- Where is resistance?
- What's the breakthrough?

### 5. Verify Arc Logic
- Is change believable?
- Is it earned through struggle?
- Does it connect to theme?
- Is pacing appropriate?

### 6. Check Arc Interactions
- How do character arcs affect each other?
- Do relationships drive growth?
- Are arcs complementary or contrasting?

### 7. Document Arc Map
Create visual or table showing:
- Character arc progression by chapter
- Key transformation moments
- Arc intersections

### 8. Update Story Bible
- Add arc details to character sections
- Note key arc scenes in timeline

## Output
- Character arc map document
- Updated Story Bible
