# Create Story Bible Workflow

## Objective
Initialize and populate the Story Bible as the single source of truth for all story details.

## Prerequisites
- Story Requirements Document (SRD) completed
- Basic story concept defined

## Steps

### 1. Initialize Story Bible
- Copy story-bible-template.md to output folder
- Fill in project information from SRD
- Set initial update date

### 2. Character Foundation
- Create entries for all known main characters
- Add basic physical descriptions and roles
- Document initial relationships
- Note character goals and motivations

### 3. World Foundation
- Define primary setting and time period
- Create entries for key locations
- Establish world rules (magic, technology, social)
- Begin sensory palette

### 4. Plot Foundation
- Document main plot thread
- Note known subplots
- Create initial timeline structure
- Mark any foreshadowing planned

### 5. Style Establishment
- Define POV and tense
- Establish narrative tone
- Note any voice conventions

### 6. Save and Reference
- Save to `{output_folder}/story-bible.md`
- All agents should reference this document
- Update after each chapter

## Output
- `{output_folder}/story-bible.md`

## Notes
- Story Bible grows with the story
- Update immediately when new details are established
- Flag contradictions for resolution
