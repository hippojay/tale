# Create Story Structure Workflow

## Objective
Design complete story structure from premise to resolution using proven frameworks.

## Prerequisites
- SRD completed
- Genre identified

## Steps

### 1. Choose Framework
Based on genre and story type:
- Three-Act Structure (universal)
- Hero's Journey (adventure, fantasy)
- Save the Cat (commercial fiction)
- Seven-Point Structure (plot-driven)
- Kishotenketsu (literary, quiet stories)

### 2. Define Core Elements
- **Premise:** One-sentence story summary
- **Central Question:** What keeps readers reading?
- **Stakes:** What's at risk?
- **Theme:** What's the story really about?

### 3. Structure Act One (25%)
- Opening image/hook
- Establish normal world
- Inciting incident
- Debate/refusal
- First plot point/threshold

### 4. Structure Act Two (50%)
- New world/challenges
- Rising action and complications
- Midpoint shift/revelation
- Consequences and escalation
- Dark moment/all is lost

### 5. Structure Act Three (25%)
- Final push/gathering
- Climax
- Resolution
- Final image

### 6. Map Subplots
- Identify 2-3 subplots
- Align with main plot beats
- Ensure they enhance, not distract

### 7. Verify Genre Compliance
- Check against genre expectations
- Ensure required elements present
- Balance convention with originality

### 8. Document Structure
- Create structure document
- Update Story Bible with plot section
- Prepare for chapter planning

## Output
- Story structure document
- Updated Story Bible
