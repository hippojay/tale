# Magic System Workflow

## Objective
Design consistent magic or technology system.

## Steps

### 1. Core Concept
- What is the source?
- Who can use it?
- How is it accessed?

### 2. Rules
- What can it do?
- What can't it do?
- What are the costs/limitations?

### 3. Consequences
- How does it affect society?
- Economic implications?
- Political implications?

### 4. Learning/Access
- How is it learned?
- Can anyone learn?
- What determines power level?

### 5. Costs
- Physical cost?
- Resource cost?
- Moral/ethical cost?

### 6. Limitations
- What stops it being all-powerful?
- Weaknesses?
- Countermeasures?

### 7. Integration
- How does it fit story?
- How do characters use it?
- Plot implications?

### 8. Document
- Clear rules in Story Bible
- Examples of use
- Limitations clearly stated

## Output
- Magic/tech system document
- Updated Story Bible
