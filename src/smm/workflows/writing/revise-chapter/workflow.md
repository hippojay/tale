# Revise Chapter Workflow

## Objective
Revise existing chapter based on editorial feedback.

## Prerequisites
- Original chapter
- Review/editorial feedback
- Story Bible

## Steps

### 1. Review Feedback
- Read all feedback completely
- Categorize by type (structure, character, prose)
- Prioritize by impact
- Identify dependencies

### 2. Plan Revisions
- List specific changes needed
- Determine order of operations
- Note potential ripple effects
- Identify what to preserve

### 3. Structural Revisions
If needed:
- Add/remove scenes
- Reorder content
- Adjust pacing
- Strengthen transitions

### 4. Character Revisions
If needed:
- Clarify motivation
- Strengthen voice
- Deepen reactions
- Fix consistency issues

### 5. Scene-Level Revisions
- Strengthen weak scenes
- Enhance conflict
- Improve dialogue
- Balance description

### 6. Prose Revisions
- Tighten language
- Vary sentence structure
- Eliminate repetition
- Sharpen word choices

### 7. Preserve Strengths
- Don't fix what works
- Maintain voice
- Keep effective passages
- Preserve emotional beats

### 8. Continuity Check
- Verify against Story Bible
- Check for new inconsistencies
- Update Story Bible if needed

### 9. Quality Verification
- Read revised chapter aloud
- Check flow and rhythm
- Verify feedback addressed
- Ensure no new problems

### 10. Document Changes
- Note significant revisions
- Flag for re-review if major
- Update any tracking documents

## Output
- Revised chapter
- Change summary
- Updated Story Bible (if needed)
