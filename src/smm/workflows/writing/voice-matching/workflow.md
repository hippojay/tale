# Voice Matching Workflow

## Objective
Analyze and match established narrative voice for consistency.

## Prerequisites
- Existing chapters with established voice
- Story Bible style guide

## Steps

### 1. Analyze Existing Voice
Read established chapters noting:
- Sentence length patterns
- Vocabulary level
- Tone (formal/casual/lyrical)
- POV depth (close/distant)
- Narrative intrusion level

### 2. Document Voice Elements

**Sentence Structure:**
- Average length
- Complexity level
- Fragment usage
- Run-on patterns

**Word Choice:**
- Vocabulary sophistication
- Jargon/slang usage
- Metaphor style
- Adjective density

**Tone:**
- Serious/humorous
- Dark/light
- Cynical/optimistic
- Formal/conversational

**POV Characteristics:**
- Interiority depth
- Sensory focus
- Emotional transparency
- Narrative distance

### 3. Identify Voice Markers
- Recurring phrases
- Characteristic constructions
- Rhythm patterns
- Unique stylistic choices

### 4. Create Voice Guide
Document for reference:
- Do's and don'ts
- Example passages
- Characteristic sentences
- Words to use/avoid

### 5. Test Voice Match
- Write sample paragraph
- Compare to established voice
- Adjust as needed
- Verify consistency

### 6. Apply to New Content
- Reference voice guide while writing
- Check periodically against examples
- Maintain consistency
- Flag any intentional variations

### 7. Update Story Bible
- Add voice guide to style section
- Include example passages
- Note any evolution in voice

## Output
- Voice analysis document
- Voice guide for reference
- Updated Story Bible style section
