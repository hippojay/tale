# Write Scene Workflow

## Objective
Write individual scene with specific objectives.

## Prerequisites
- Scene outline/beats
- Story Bible for continuity
- Previous scene for transition

## Steps

### 1. Scene Setup
- Identify POV character
- Establish location
- Note characters present
- Define scene objective
- Identify conflict source

### 2. Opening
- Transition from previous scene (if applicable)
- Ground reader in time/place
- Establish POV quickly
- Hook into scene conflict

### 3. Scene Development
- Build tension through conflict
- Include sensory grounding
- Balance dialogue/action/interiority
- Show character reactions
- Advance toward scene goal

### 4. Dialogue
- Keep it purposeful
- Reveal character through speech
- Include subtext
- Use minimal tags
- Break with action beats

### 5. Interiority
- Show POV character's thoughts
- Connect to emotional arc
- Don't over-explain
- Balance with external action

### 6. Scene Climax
- Deliver on scene promise
- Create change or revelation
- Impact character emotionally
- Advance plot

### 7. Closing
- Resolve immediate tension (or not)
- Set up transition
- End with hook or question
- Leave reader wanting more

### 8. Quality Check
- Does scene have clear purpose?
- Is conflict present throughout?
- Is POV consistent?
- Does it advance story?
- Is pacing appropriate?

### 9. Continuity Verification
- Check against Story Bible
- Verify character details
- Confirm timeline
- Note any new facts to add

## Output
- Completed scene
- Notes for Story Bible updates
